Main


Schon als Kind habe ich mich für Computer interessiert und wie sie funktionieren. So habe ich schon früh mit der grafischen Programmiersprache Scratch angefangen und konnte so schon meine ersten Spiele programmieren. In der Schule habe ich dann mein Programmier Wissen erweitert und eine einfache Version von JavaScript gelernt. Damit ist es mir gelungen, das Spiel Space Invaders nach zu stellen.

Auch mein P-Seminar habe ich der Informatik gewidmet und arbeite so mit an unserer Schüler Website der Bapp. Zu Hause habe ich mir dann die Programmiersprachen HTML, JavaScript und CSS beigebracht, mit denen ich auch diese Website erstellt habe.

Künstliche Intelligenz


Zusätzlich interessiere ich mich für Künstliche Intelligenzen und benutze diese auch oft in meinem Altag. Wenn man weiß, wie man diese richtig einsetzt, dann kann man sich eine Menge an Arbeit ersparen. Sie sind ein Sinnvolles Werkzeug, von dessen Gebrauch ich überzeugt bin. Dadurch bin ich auf ComfyUi (Stable Diffusion) gekommen.Mit dieser KI habe ich die Möglichkeit allerlei Bilder zu erstellen. Ich habe schon viele Projekte mit dieser KI erstellt, inklusive ein paar Videos. Ein paar Bilder habe ich auch unten als Beispiele mit hochgeladen.

Website


Diese Website ist Produkt aus einer meiner Ideen. Mit täglicher arbeit über Wochen verteilt habe ich die erste Version meiner Website vollständigt. An diesem Projekt zu arbeiten, wird wohl nie aufhören, denn es gibt immer was zu tun. Sie soll ein Beispiel für mich darstellen, eine Vorstellung über mich als Person geben und ich bin Stolz darauf, wie weit ich schon damit gekommen bin.
